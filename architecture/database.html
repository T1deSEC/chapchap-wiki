Flyway 마이그레이션

버전 파일명 내용
V1 V1__users.sql 유저 도메인 전체 — users, user_skin_profile, user_skin_concern, user_ingredient_preference, user_routine, user_routine_product, user_daily_skin_log, user_daily_skin_log_keyword, refresh_tokens, user_notification_settings + set_updated_at 트리거
V2 V2__products.sql products, ingredients, product_ingredients
V3 V3__features.sql user_wishlist, product_feedback, notifications, user_ingredient_recommendation, user_recommended_ingredient, user_avoid_ingredient, user_recommended_product, user_product_ai_analysis, user_product_ai_ingredient
V4 V4__real_products.sql 실제 화장품·성분 시드 데이터 (485KB, products/ingredients/product_ingredients 대량 INSERT)
정책: V1~V3은 DDL(스키마) 전용입니다. V4는 실제 제품 시드 데이터. 추가 마이그레이션은 V5부터 작성합니다.

유저 도메인

users

컬럼 타입 설명
user_idBIGINTPK (GENERATED BY DEFAULT AS IDENTITY)
emailVARCHAR(255)NOT NULL, UNIQUE
password_hashVARCHAR(255)NOT NULL (BCrypt 해시)
nicknameVARCHAR(50)NOT NULL
created_atTIMESTAMPNOT NULL, 가입 일시
updated_atTIMESTAMPNOT NULL, 트리거 자동 갱신

user_skin_profile

컬럼 타입 설명
user_idBIGINTPK, FK → users.user_id (CASCADE)
skin_typeVARCHAR(30)NOT NULL (건성/지성/복합성/민감성 등)
genderVARCHAR(20)nullable
birth_yearSMALLINTnullable
updated_atTIMESTAMPNOT NULL, 트리거 자동 갱신

user_skin_concern

컬럼 타입 설명
user_idBIGINT복합 PK, FK → users.user_id (CASCADE)
concern_codeVARCHAR(30)복합 PK, 피부 고민 코드

user_ingredient_preference

컬럼 타입 설명
user_idBIGINT복합 PK, FK → users.user_id (CASCADE)
ingredient_idBIGINT복합 PK
preference_typeVARCHAR(20)NOT NULL (prefer / avoid)
updated_atTIMESTAMPNOT NULL, 트리거 자동 갱신

user_routine

컬럼 타입 설명
routine_idBIGINTPK
user_idBIGINTFK → users.user_id (CASCADE)
routine_periodVARCHAR(10)NOT NULL — AM 또는 PM
record_dateDATENOT NULL — 서버에서 today 자동 설정
UNIQUE(user_id, routine_period) — 사용자당 AM/PM 각 1개

user_routine_product

컬럼 타입 설명
routine_idBIGINT복합 PK, FK → user_routine.routine_id (CASCADE)
product_idBIGINT복합 PK
step_orderSMALLINTNOT NULL — 루틴 내 적용 순서, UNIQUE(routine_id, step_order)

user_daily_skin_log

컬럼 타입 설명
skin_log_idBIGINTPK
user_idBIGINTFK → users.user_id (CASCADE)
log_dateDATENOT NULL, UNIQUE(user_id, log_date) — 날짜당 1개
skin_statusVARCHAR(20)NOT NULL — 당일 피부 상태
memoTEXT자유 메모
photo_urlVARCHAR(500)피부 사진 URL
am_routine_idBIGINTFK → user_routine (SET NULL)
pm_routine_idBIGINTFK → user_routine (SET NULL)
am_executedBOOLEANNOT NULL, DEFAULT FALSE
pm_executedBOOLEANNOT NULL, DEFAULT FALSE

user_daily_skin_log_keyword

컬럼 타입 설명
skin_log_idBIGINT복합 PK, FK → user_daily_skin_log (CASCADE)
keyword_codeVARCHAR(30)복합 PK — 피부 상태 키워드 코드

refresh_tokens

컬럼 타입 설명
token_idBIGINTPK
user_idBIGINTFK → users.user_id (CASCADE)
token_hashVARCHAR(255)NOT NULL, UNIQUE
expires_atTIMESTAMPNOT NULL — 만료 일시
created_atTIMESTAMPNOT NULL

user_notification_settings

컬럼 타입 설명
user_idBIGINTPK, FK → users.user_id (CASCADE)
ingredient_analysis_enabledBOOLEANNOT NULL, DEFAULT TRUE
ingredient_recommend_enabledBOOLEANNOT NULL, DEFAULT TRUE
routine_analysis_enabledBOOLEANNOT NULL, DEFAULT TRUE
updated_atTIMESTAMPNOT NULL, 트리거 자동 갱신

제품·성분 도메인

products

컬럼 타입 설명
product_idBIGINTPK
nameVARCHAR(200)NOT NULL
brandVARCHAR(100)NOT NULL
categoryVARCHAR(50)NOT NULL (serum / moisturizer / toner 등)
image_urlVARCHAR(500)제품 이미지 URL
created_atTIMESTAMPNOT NULL

ingredients

컬럼 타입 설명
ingredient_idBIGINTPK
inci_nameVARCHAR(200)NOT NULL, UNIQUE — INCI 국제 성분명
ko_nameVARCHAR(200)한국어 성분명
descriptionTEXT성분 설명

product_ingredients

컬럼 타입 설명
product_idBIGINT복합 PK, FK → products (CASCADE)
ingredient_idBIGINT복합 PK, FK → ingredients (RESTRICT)
concentration_orderSMALLINTNOT NULL — 전성분 표기 순서 (낮을수록 고함량)

기능성 테이블

user_wishlist

컬럼 타입 설명
user_idBIGINT복합 PK, FK → users (CASCADE)
product_idBIGINT복합 PK, FK → products (CASCADE)
added_atTIMESTAMPNOT NULL — 찜 등록 일시

product_feedback

컬럼 타입 설명
feedback_idBIGINTPK
user_idBIGINTFK → users (CASCADE), UNIQUE(user_id, product_id)
product_idBIGINTFK → products (CASCADE)
is_effectiveBOOLEANNOT NULL — reaction으로 파생 (good→true)
reactionVARCHAR(30)good / neutral / trouble
ratingSMALLINT별점
usage_periodVARCHAR(20)사용 기간
memoTEXT자유 메모
created_atTIMESTAMPNOT NULL

notifications

컬럼 타입 설명
notification_idBIGINTPK
user_idBIGINTFK → users (CASCADE)
typeVARCHAR(30)NOT NULL — 알림 유형
titleVARCHAR(100)NOT NULL — 알림 제목
bodyTEXT알림 본문
is_readBOOLEANNOT NULL, DEFAULT FALSE
created_atTIMESTAMPNOT NULL

성분 추천 도메인

AI 성분 추천 결과를 저장하는 테이블 그룹. user_ingredient_recommendation이 루트이며, 추천 성분·회피 성분·추천 제품이 이를 참조한다.

user_ingredient_recommendation

컬럼 타입 설명
user_idBIGINTPK, FK → users (CASCADE)
summaryTEXTNOT NULL — AI 생성 추천 요약
created_atTIMESTAMPNOT NULL

user_recommended_ingredient

컬럼 타입 설명
user_idBIGINT복합 PK, FK → user_ingredient_recommendation (CASCADE)
ingredient_idBIGINT복합 PK, FK → ingredients (RESTRICT)
reasonTEXTNOT NULL — 추천 이유
sort_orderSMALLINTNOT NULL — 추천 순위

user_avoid_ingredient

컬럼 타입 설명
user_idBIGINT복합 PK, FK → user_ingredient_recommendation (CASCADE)
ingredient_idBIGINT복합 PK, FK → ingredients (RESTRICT)
reasonTEXTNOT NULL — 회피 이유

user_recommended_product

컬럼 타입 설명
user_idBIGINT복합 PK, FK → user_ingredient_recommendation (CASCADE)
product_idBIGINT복합 PK, FK → products (CASCADE)
match_scoreDECIMAL(6,2)NOT NULL — 성분 매칭 점수
rank_orderSMALLINTNOT NULL — 추천 순위

AI 분석 도메인

user_product_ai_analysis

컬럼 타입 설명
idBIGINTPK
user_idBIGINTFK → users (CASCADE), UNIQUE(user_id, product_id)
product_idBIGINTFK → products (CASCADE)
safety_scoreINTNOT NULL — 종합 안전 점수 (0~100)
summaryTEXTAI 생성 분석 요약
recommendationsTEXT[]AI 생성 사용 권고 사항 배열
skin_impactsJSONB11개 피부 카테고리별 영향 점수
created_atTIMESTAMPNOT NULL — 분석 수행 일시

user_product_ai_ingredient

컬럼 타입 설명
idBIGINTPK
analysis_idBIGINTFK → user_product_ai_analysis (CASCADE)
inci_nameVARCHAR(200)NOT NULL — 분석 시점 INCI명 스냅샷
ko_nameVARCHAR(200)한국어 성분명
safety_levelVARCHAR(20)NOT NULL — safe / caution / danger
assessmentVARCHAR(500)NOT NULL — AI 평가 요약
reasonTEXT상세 근거

테이블 관계 (ERD)

users
  ├── user_skin_profile              (1:1)
  ├── user_skin_concern              (1:N)
  ├── user_ingredient_preference     (1:N)
  ├── user_notification_settings     (1:1)
  ├── refresh_tokens                 (1:N)
  ├── user_daily_skin_log            (1:N, UNIQUE log_date)
  │     └── user_daily_skin_log_keyword (1:N)
  ├── user_routine                   (1:2, UNIQUE routine_period AM/PM)
  │     └── user_routine_product     (1:N, step_order)
  ├── user_wishlist                  (N:M → products)
  ├── product_feedback               (N:M → products, reaction/rating/usage_period)
  ├── notifications                  (1:N)
  ├── user_ingredient_recommendation (1:1)
  │     ├── user_recommended_ingredient (1:N → ingredients)
  │     ├── user_avoid_ingredient       (1:N → ingredients)
  │     └── user_recommended_product    (1:N → products)
  └── user_product_ai_analysis       (N:M → products, UNIQUE)
        └── user_product_ai_ingredient (1:N)

products
  └── product_ingredients            (N:M → ingredients, concentration_order)
