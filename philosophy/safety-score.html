AI 설계 철학

안전도 점수 계산

LLM의 주관적 추정을 코드가 결정론적으로 변환하는 원리

역할 분리 원칙

안전도 계산은 두 단계로 분리되어 있습니다.

담당 역할 특성
LLM 성분별 concern_score (0–100) 추정 비결정론적 (호출마다 미세 변동)
코드 임계값 매핑 + 농도가중 집계 결정론적 (동일 입력 → 동일 출력)
LLM 변동성을 수치 추정 단계로만 한정하고, 나머지 연산을 코드가 고정 처리합니다. 같은 분석 결과가 입력되면 항상 같은 safety_score가 출력됩니다.

1단계: safety_level 산출

LLM이 추정한 concern_score를 고정 임계값으로 세 수준으로 변환합니다.

concern_score < 34 → 안전
34 ≤ concern_score < 67 → 주의
concern_score ≥ 67 → 위험

2단계: safety_score 산출 (농도가중 평균)

각 성분의 점수를 함량 가중치로 집계합니다.

// 1. 성분별 안전 점수 계산
score_i = clamp(100 − concern_score_i × multiplier(assessment_i), 0, 100)

// assessment → multiplier 매핑
적합 → multiplier = 0.5
주의 → multiplier = 1.0
부적합 → multiplier = 1.5

// 2. 함량 가중치 (고함량 성분에 더 큰 비중)
weight_i = 1 / concentration_order_i

// 3. 최종 가중 평균
safety_score = round( Σ(weight_i × score_i) / Σ(weight_i) )

계산 예시

3개 성분을 가진 제품:

성분 concern_score assessment order score_i weight_i
히알루론산 10 적합 1 clamp(100−10×0.5) = 95 1/1 = 1.00
레티놀 50 주의 5 clamp(100−50×1.0) = 50 1/5 = 0.20
변성 에탄올 80 부적합 10 clamp(100−80×1.5) = 0 1/10 = 0.10
safety_score = round( (95×1.00 + 50×0.20 + 0×0.10) / (1.00 + 0.20 + 0.10) )
= round( (95 + 10 + 0) / 1.30 )
= round( 80.77 )
= 81

설계 선택의 이유

  • 결정론성 보장 — 같은 분석 결과를 두 번 읽어도 UI에 같은 숫자가 표시됩니다.
  • 농도 가중치 — 0.1% 미만 미량 성분이 safety_score를 크게 끌어내리는 것을 방지합니다.
  • multiplier 세 단계 — assessment(적합/주의/부적합)에 따라 concern_score의 영향력을 조절합니다. "부적합이지만 concern 낮음"과 "적합이지만 concern 높음"을 서로 다르게 다룹니다.
