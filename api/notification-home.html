알림 & 홈

알림: 인증 필요 / 홈 날씨 팁: 공개 경로

알림

GET/api/notifications

알림 목록 조회 (페이지네이션). 응답 타입: PageResponse<NotificationResponse>

GET /api/notifications?page=0&size=20
// Response data
{
  "content": [
    {
      "id": 1,
      "type": "ROUTINE_CONFLICT",
      "title": "루틴 충돌 감지",
      "body": "AM 루틴의 레티놀과 비타민 C 성분이 충돌합니다.",
      "isRead": false,
      "createdAt": "2026-06-11T10:00:00"
    }
  ],
  "totalElements": 5,
  "totalPages": 1,
  "number": 0,
  "size": 20
}

알림 타입 4종

type 발생 시점
INGREDIENT_ANALYSISAI 성분 분석 완료 시
INGREDIENT_RECOMMENDATIONAI 성분 추천 완료 시
ROUTINE_CONFLICT루틴 분석 결과 conflict 시
ROUTINE_CAUTION루틴 분석 결과 caution 시

GET/api/notifications/unread-count

미읽음 알림 수 조회.

// Response: ApiResponse<Long>
// unwrap 후 r.data가 숫자 직접값
{ "success": true, "data": 3, "message": null }
⚠ r.data가 숫자 직접값 — unwrap 후 r.data가 바로 3입니다. r.data.count로 접근하면 undefined가 됩니다.

PUT/api/notifications/:id/read

알림 단건 읽음 처리.

PUT/api/notifications/read-all

모든 알림 읽음 처리.

홈 날씨 팁

공개 경로 — 인증 불필요. 동일 좌표 요청은 서버에서 1시간 캐시 (@Cacheable).

GET/api/home/weather-tip

현재 날씨 기반 피부 케어 팁. 기본값은 서울 좌표.

GET /api/home/weather-tip?lat=37.5665&lon=126.9780
