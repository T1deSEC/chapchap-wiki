API 레퍼런스

제품 & 성분

공개 경로 — 인증 불필요: /api/products/**, /api/ingredients/**

모든 응답은 ApiResponse<T> 래퍼. 프론트 interceptor 자동 언래핑.

제품

GET/api/products

제품 검색. 응답 타입: PageResponse<ProductSummary>

GET /api/products?search=세럼&category=SERUM&page=0&size=20
// Response data
{
  "content": [
    {
      "id": 1,
      "name": "레티놀 세럼",
      "brand": "BRAND A",
      "category": "SERUM",
      "imageUrl": "https://...",
      "ingredientCount": 24
    }
  ],
  "totalElements": 152,
  "totalPages": 8,
  "number": 0,
  "size": 20
}
실제 배열은 .content에 있습니다. 훅에서 .then(r => r.data.content)로 추출해야 합니다.

GET/api/products/:id

제품 상세 조회. 성분 목록(concentration_order 순) 포함.

// Response data
{
  "id": 1,
  "name": "레티놀 세럼",
  "brand": "BRAND A",
  "category": "SERUM",
  "imageUrl": "https://...",
  "ingredients": [
    {
      "inciName": "RETINOL",
      "koName": "레티놀",
      "concentrationOrder": 5,
      "functionTags": []
    }
  ]
}
⚠ safetyScore 없음 — 제품 상세 응답에 safetyScore 필드가 없습니다. 프론트엔드에서 ingredients의 각 성분 AI 분석 결과를 집계하여 computeSafetyScore(ingredients)로 로컬 계산합니다.

성분

GET/api/ingredients

성분 목록 검색.

GET /api/ingredients?search=레티놀
// Response data (array)
[
  {
    "id": 42,
    "inciName": "RETINOL",
    "koName": "레티놀",
    "description": "비타민 A 유도체. 세포 재생 및 항노화 효과."
  }
]

GET/api/ingredients/:id

성분 단건 상세 조회.

// Response data
{
  "id": 42,
  "inciName": "RETINOL",
  "koName": "레티놀",
  "description": "비타민 A 유도체. 세포 재생 및 항노화 효과."
}
⚠ safety_level, function_tags 없음 — V11·V12 마이그레이션에서 삭제되었습니다. 성분 안전도는 AI 분석(POST /api/analysis/ingredient) 결과로만 제공됩니다.

concentration_order 의미

화장품 전성분 표기는 함량 높은 성분부터 오름차순으로 기재합니다. concentration_order 값이 작을수록 제품 내 함량이 높습니다.
예: concentration_order=1은 가장 많이 들어간 성분, concentration_order=30은 미량 성분.

이 값은 match_score 계산에서 가중치로 사용됩니다. 자세한 내용은 성분 랭킹 알고리즘을 참고하세요.
