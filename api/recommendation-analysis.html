AI 성분 분석 (LangGraph)

POST/api/analysis/ingredient

제품 성분 AI 분석 실행 및 결과 DB 저장.

// Request body
{
  "productId": 1,
  "userSkinType": "DRY",
  "userSkinConcerns": ["ACNE", "SENSITIVITY"]
}

GET/api/analysis/ingredient/:productId

저장된 분석 결과 조회 (캐시). 분석 전이면 404.

// Response data
{
  "safetyScore": 72,
  "ingredientAnalysis": [
    {
      "inciName": "RETINOL",
      "koName": "레티놀",
      "assessment": "주의",
      "reason": "민감성 피부에 자극 가능성. 저농도 사용 권장.",
      "concentrationOrder": 5
    }
  ],
  "summary": "전반적으로 안전하나 레티놀 함량에 주의 필요.",
  "recommendations": ["야간 루틴에만 사용", "주 2~3회부터 시작"],
  "skinImpacts": {
    "보습": 85,
    "항노화·재생": 78,
    "피부 장벽": 60,
    "진정·항염": 40
  }
}

safetyScore 기준

점수 범위 레벨 UI 표시
≥ 60높음 (primary)녹색
30 ~ 59중간 (warning)노란색
< 30낮음 (danger)빨간색

skinImpacts 카테고리 (11개)

보습 피부 장벽 미백·톤업 항산화 진정·항염 각질케어 항노화·재생 탄력 피지·모공 항균·트러블케어 자외선 차단

AI 루틴 분석 (LangGraph)

POST/api/analysis/routine

루틴 제품 간 성분 충돌 분석.

// Request body
{
  "productIds": [1, 5, 12],
  "routinePeriod": "AM",
  "userSkinType": "DRY"
}
// Response data
{
  "status": "conflict",
  "conflictingPairs": [
    {
      "productId1": 1,
      "productId2": 5,
      "conflictIngredient": "RETINOL",
      "reason": "레티놀과 비타민 C 동시 사용 시 자극 증가 및 효능 저하"
    }
  ],
  "recommendation": "레티놀 제품은 PM 루틴으로 이동하세요.",
  "suggestedAdjustments": [
    { "productId": 1, "action": "move_to_pm" }
  ]
}
알림 자동 트리거
status: "conflict" → ROUTINE_CONFLICT 알림 발송
status: "caution" → ROUTINE_CAUTION 알림 발송
status: "good" → 알림 없음
